fit

  • fit(options: object): Promise<boolean>
    • Defined in sequential.ts:171

    Trains the model for a fixed number of epochs: iterations on the dataset.

    Parameters

    • options: object
      • Optional batchSize?: number

        Number of samples per gradient update. If unspecified, it will default to 32.

      • Optional epochs?: number

        Number of epochs to train. Each epoch is an iteration over the entire input and target data provided. If unspecified, it will default to 1.

      • input: any[]

        Array of training data. Mandatory.

      • Optional log?: number

        Number of epochs to run between logging. 0 = silent. 10 = log every 10 epochs. If unspecified, it will default to 10.

      • Optional shuffle?: boolean

        Boolean (whether to shuffle the training data before each epoch). If unspecified, it will default to true.

      • target: any[]

        Array of target (label) data. Mandatory.

      • Optional validationSplit?: number

        TO BE IMPLEMENTED. Float between 0 and 1. Fraction of the training data to be used as validation data. The model will set apart this fraction of the training data, will not train on it, and will evaluate the loss and any model metrics on this data at the end of each epoch. The validation data is selected from the last samples in the x and y data provided, before shuffling. Default: 0 == no validation.

    Returns Promise<boolean>

    Promise, resolving to true once the model is trained.
